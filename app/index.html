Model 3

Окунитесь в мир умопомрачительного разгона и управляемости вместе с самой популярной моделью Tesla - Model 3!

Ураганный разгон c 0 - 100 км/ч за считанные 3.3 секунды оставит далеко позади даже самые мощные спорткары с бензиновыми двигателями.

Система полного привода Tesla All-Wheel Drive имеет два независимых двигателя для улучшения эффективности, каждый из которых имеет только одну движущуюся часть что ведет к минимальному износу и максимальной долговечности.

Внутреннее пространство Model 3 не похоже ни на один другой автомобиль. Вы можете использовать свой смартфон в качестве ключа и получить доступ ко всем органам управления водителя на центральном 15-дюймовом сенсорном экране. Полностью стеклянная крыша простирается спереди назад, создавая ощущение открытости с каждого сиденья.

Model Y
